# FreshTrack — Career & Habit OS

A mobile-first productivity app for freshers job seekers to track habits, job applications, skills, and career progress.

## Features

- **Dashboard** - Overview of daily progress, streaks, and KPIs
- **Habits** - Track daily habits with categories and completion tracking
- **Focus Timer** - Pomodoro timer for focused work sessions
- **Job Tracker** - Track job applications through pipeline stages
- **Skills** - Track skill proficiency levels
- **Resume Prep** - Checklist and tips for resume building
- **Interview Prep** - Question bank and practice answers
- **Daily Notes** - Quick notes and tasks

## iOS-Style Mobile App

- Bottom tab navigation (Home, Dashboard, Career, Workflow)
- Mobile-optimized layout
- PWA support - install as native app

## Tech Stack

- Vanilla JavaScript
- CSS3 with iOS styling
- HTML5

## Usage

Open `index.html` in a browser. Works offline when installed as PWA.
